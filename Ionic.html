Ionic

¿Qué es Ionic?

Ionic es un framework de código abierto el cual permite construir aplicaciones móviles utilizando HTML, CSS y JavaScript. Es compatible con múltiples plataformas (Android, iOS Web)

Configuración

instalación guiada por la página de Ionic

1. Abrir terminal y ejecutar: 

npm install -g @ionic/cli

2. Crear un Nuevo Proyecto

Crear un nuevo proyecto usando el comando "ionic start" 

ionic start nombreDelProyecto

3. Navegar al Directorio del Proyecto

Acceso al proyecto: 

cd nombreDelProyecto

4. Ejecutar la Aplicación en el Navegador

Ejecuta: 

ionic serve

5. Configurar Plataforma (Opcional)

Para Android: 

ionic cap add android

Para iOS: 

ionic cap add ios

Luego, abrir el proyecto correspondiente a la plataforma seleccionada

Android: 

ionic cap open android

iOS: 

ionic cap open ios

Carácteristicas

1. Base de código: Desarrollo de aplicaciones para múltiples plataformas (iOS, Android, Web)

2. Componentes de Interfaz de Usuario (UI) Ofrece una amplia colección de componentes de interfaz de usuario predefinidos, además proporciona una experiencia de usuario consistente y nativa.

3. Compatibilidad Ionic permite crear aplicaciones que se ejecutan en iOS, Android y la web desde una sola base de código.

4. Integración con Frameworks Compatibilidad con frameworks como Angular, React y Vue.js.

5.Escalabilidad y Flexibilidad Permite crear aplicaciones de simples ha complejas con facilidad.

6. Documentación y Comunidad Activa: Ionic tiene documentación detallada lo cual permite que el desarrollo y entender el manejo del desarrollo sea más facil.

7. Herramientas de CLI

Incluye interfaz de línea de comandos (CLI) facilidad para creación y prueba de aplicaciones.
