Typescript

¿Qué es?

TypeScript es un lenguaje de programación de tipado estático, Permite agregar tipos a variables, funciones y objetos, permitiendo al compilador detectar errores en tiempo de desarrollo.

Sintaxis

Asignación

Se utiliza el símbolo = para asignar un valor a una variable

Tipos (:)

Permite definir el tipo de variable, valor de retorno y el parámetro.

Símbolos

Únicamente se permite _ y $

Tipos de Unión (Union Types)

Facilita tener valores de diferentes tipos.

Unicidad

Deben ser únicos.

Enumeraciones (Enums)

permite definir un conjunto de valores con nombre

Declaración de Variables

(let, const, var): let y const sirven para declarar variables. let permite cambiar el valor y const define una constante.

Definición de varibales

Variables

Son contenedores para almacenar datos

- let: Declara variables cuyo valor puede cambiar
- var: Su uso no es recomendable, ya que provoca problemas con el hoisting (se elevan al comienzo de su ámbito). - const: Declara constantes, es decir, variables cuyo valor no puede cambiar

Tipos

Es una estructura que se puede utilizar en una aplicación. TypeScript es un superconjunto de JavaScript que agrega tipado estático, Permite detectar a tiempo errores de compilación.

Clases (class)

Se usa para definir clases usando modificadores como public, private, y protected

Interfaces

interface: Permite definir la forma que debe tener un objeto

Métodos

Ofrece una amplia gama de métodos como:

1. forEach()

Ejecuta una función para cada elemento del array.

2. pop()

Elimina el último elemento del array y lo devuelve.

3. unshift()

Agrega uno o más elementos al principio del array y devuelve la nueva longitud del array

4. push()

Agrega elementos al final del array y devuelve la nueva longitud del array.

5. includes()

Determina si un array incluye un determinado elemento, devolviendo true o false.

6. filter()

Crea un nuevo array con elementos que cumplan una condición.

7. splice()

Añade, elimina o reemplaza elementos dentro de un array.

8. map()

Crea un nuevo array con los resultados de aplicar una función a cada elemento del array original
