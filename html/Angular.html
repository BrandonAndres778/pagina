Angular

¿Qué es?

Es un framework de desarrollo de aplicaciones web de código abierto, permite la creación de pequeñas a grandes aplicaciones, mediante el uso de TypeScript, este framework utiliza un enfoque basado en componentes.

Características

Componentes: permite a las aplicaciones dividir la interfaz de usuario en partes reutilizables y manejables.

CLI: Permite automatizar tareas como la configuración de proyectos.

Enrutamiento: Permite la navegación entre diferentes vistas y componentes de manera eficiente, lo que facilita la navegación en aplicaciones de una sola página.

Inyección de Dependencias: Facilita la creación de servicios que pueden ser utilizados en diferentes partes de la aplicación mediante la modularidad y escalabilidad

RxJS:Utiliza RxJS para manejar programación reactiva, lo que facilita la gestión de eventos asíncronos.

RxJS:Utiliza RxJS para manejar programación reactiva,facilitando la gestión de eventos asíncronos.

TypeScript: Lenguaje con tipado estático y estructura.

Configuración

Para la creación de un proyecto en Angular, debes de llevar a cabo los siguientes pasos:

1.Instalación en cmd: npm install -g @angular/cli

2.Ejecutar el comando para crear un nuevo proyecto: ng new nombre-del-proyecto

3.Navegar al directorio del proyecto: cd nombre-del-proyecto

4.Da inicio: ng serve

5.Abre tu navegador y verifica el funcionamiento: "http://localhost:4200/"

Signal

En Angular se refiere a una forma de manejar el estado la aplicación. Aunque no es una característica oficial de Angular, Lo cual facilita que un valor cambie con el tiempo y automáticamente, actualizando cualquier parte de la aplicación que dependa de ese valor, logrando mejorar el rendimiento y la reactividad de las aplicaciones.

1. Computed Signal:: Signal cuyo valor depende de otros Signals y se actualiza automáticamente

2. Signal Arrays and Objects: Contiene estructuras complejas como arrays y objetos

3. Effect: Ejecuta una función automáticamente cuando los Signals que usa cambian.

4. Writable Signal: Hace referencia a un "Signal" que se puede actualizar automáticamente
